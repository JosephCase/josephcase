Positioning

I have used the CSS3 position property in different ways.

The main navigation bar has been given a position value of fixed, so that it is fixed in the same position on the screen, even when scrolling.

It's been given a width value of 100% so that it spans the width of the screen for all resolutions.

The pages themselves have been given a position value of absolute (this is later manipulated using JavaScript).

To make the content sit centrally on screens of all resolutions I have; given them a fixed width of 960px, set their left positioning value to 50% and then given them a margin-left value of -480px (half page width).

:hover

The CCS3 pseudo class property :hover is extremely useful for adding interactivity to webpages simply, without the need for JavaScript.

The navigation bar above demonstrates its usefulness for creating drop down boxes.

Notice each header's drop down menu is revealed in a different style, the latter two using the CCS3 transition property.

Shadows

I've also taken advantage of the CSS3 box-shadow property to give subtle shadows to some elements to give a more 3-dimensional-layered feel to the page.

Opacity

You will notice each drop down menu has slight transparency created using the CSS3 opacity property.

Rounded Corners

The drop down menu's have also had their bottom corners rounded using the CSS3 border-radius property.
