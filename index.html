Artist portfolio website with custom made CMS, allowing the user to create and edit pages, and upload text, image and video content.

A project developed in my own time, it consists of an independant website written in PHP and a Node.js CMS. It has a heirachal structure of pages and content stored in a mySQL database.

The key aim of the website was to provide high quality images across multiple devices, while still achieving fast load times. In order to achieve this the CMS allows the user to upload images which are then saved at multiple sizes using Imagemagick and the Imagemagick wrapper module. The website is then able to choose which image size to serve based on the display width of the image.

The Node CMS uses the Express framework to route HTTP requests, serve static content and handle user session. Multiple page edits can be made at once and the request is sent to the server as a JSON list of instructions, which are then executed asynchronously, updating the mySQL database and then performing an action on any associated files. The user is able to upload images as mentioned, but also video (which is optimised and saved in mp4 and webm formats using ffmpeg) and text. The user has control over the order and size of all content as well as the language it should appear in.

The website itself uses the Mod_rewrite engine to route all requests to a single PHP script, which serves content based on the url request using a single page class. The CMS inputted text is formatted using Markdown and the website has a simple session language toggle which controls which content is displayed based on its language setting in the CMS.
